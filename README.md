# Sunseed 3152

## Bienvenue dans le JARDIN DU FUTUR

Dans **Sunseed 3152**, vous incarnez un drone agricole **DU FUTUR**.

### Objectif
Votre objectif est de planter et de récolter **20 légumes** (ni plus, ni moins).

### Commandes

| Action | Clavier | Manette |
| :--- | :---: | :---: |
| **Déplacement** | ZQSD | Stick gauche |
| **Planter / Ramasser** | Clic gauche | Bouton Est |
| **Arroser** | Clic droit | Bouton Sud |
| **Pause** | Echap | Start |

### Caractéristiques Techniques

* **Moteur** : Unity 6 (6000.2.7f2)
* **Render Pipeline** : Universal Render Pipeline (URP)
* **Système d'Input** : Unity Input System (Support Clavier/Souris & Manette)
* **Packages Clés** :
  * Cinemachine (Caméra)
  * AI Navigation
  * TextMeshPro
* **Plateformes Cibles** : Windows, macOS, Linux, WebGL

### Installation & Développement

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/EmilPhOniX/Sunseed-3152.git
   ```
2. Ouvrir le projet avec **Unity Hub** (Version 6000.2.x recommandée).
3. Lancer la scène principale située dans `Assets/Scenes`.
